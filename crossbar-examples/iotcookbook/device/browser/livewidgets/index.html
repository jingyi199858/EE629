WAMP Live Data Widgets

With WAMP Live Data Widgets, it's easy to implement a live display of your data in the browser.

You can combine the widgets listed below to create dashboards, or integrate one or more into your Web pages as needed.

Sample Dashboard Controller for the Dashboard

To test the widgets, open the dashboard and controller on separate displays or devices. Changes entered in the controller are displayed on the dashboard in real time.
